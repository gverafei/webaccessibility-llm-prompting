Welcome!

Accessible University (AU) is a fictional university, and this is its fictional home page. This page is designed to demonstrate a variety of common web design problems that result in visitors with disabilities being unable to access the content or features of a web page. Although the heading at the top of this section of the page suggests that AU may be a welcoming institution, the inaccessible design of this page sends the opposite message.

Bienvenido!

Accesible Universidad (UA) es una universidad ficticia, y esta es su página de ficción. Esta página está diseñada para demostrar una variedad de común de diseño web que se traducen en problemas de los visitantes con discapacidad no poder acceder al contenido o las características de una página web. Aunque el título en la parte superior de esta sección de la página sugiere que AU puede ser una institución de bienvenida, el diseño inaccesible de esta página envía el mensaje opuesto.

Accessibility Issues
  1. Missing landmark regions (Addressed with header, main, nav, footer, aside)
  2. No headings (Addressed with h1, h2)
  3. Language not specified (Addressed with lang attribute)
  4. No alternate text on informative images (Addressed with alt text)
  5. Missing or excessive alternate text on decorative images (Addressed with alt="")
  6. Insufficient color contrast (Noted in CSS, requires manual check/adjustment)
  7. Color used to communicate information (Addressed with text labels and aria-required)
  8. Inaccessible keyboard interface (Addressed with Bootstrap components)
  9. No visible indication of focus (Provided by browser defaults and Bootstrap)
  10. No "skip to main content" link (Addressed with skip link)
  11. Redundant, uninformative link text (Links updated where possible, e.g., in footer)
  12. Links vs buttons (Submit button is input type=submit, modal close/ok are buttons)
  13. Inaccessible navigation menu (Addressed with Bootstrap navbar and nav with dropdowns)
  14. Inaccessible carousel (Addressed with Bootstrap carousel)
  15. Inaccessible modal dialog (Addressed with Bootstrap modal)
  16. Form not properly labeled (Addressed with label elements and for/id)
  17. Inaccessible CAPTCHA (Addressed with alt text and note about needing alternative)
  18. Inaccessible input validation (Original JS removed, validation needs accessible implementation)
  19. Missing accessible table markup (Addressed with thead, tbody, th, scope, caption)
  20. Missing abbreviation tags (Not critical for WCAG 2.2, not added)
  21. Inaccessible audio content in video (Noted need for captions/transcript)
  22. Inaccessible visual content in video (Noted need for captions/transcript)
